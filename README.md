# プライバシーポリシー

最終更新: 2026年6月9日

---

## 0. 運営者情報

本サービス「ボルスポ」は、ボルスポ運営チームが運営しています。本ポリシーに関するご質問・開示請求等は以下の窓口までご連絡ください。

お問い合わせ先: borspo.support@gmail.com

## 1. 取得する情報

本サービスは、サービス提供のため以下の情報を取得します。

- 氏名(ニックネーム)、メールアドレス、性別
- 任意で入力されたお気に入り都道府県・スポーツ・希望性別
- アップロードされた写真(プロフィール画像、チーム画像、イベント画像)
- ユーザーが入力したチーム情報・イベント情報・会場の位置情報
- チャットメッセージの内容と送受信日時
- イベント参加履歴
- 端末識別子(FCM プッシュ通知トークン)
- 利用ログ(画面遷移、ボタン操作内容)
- 位置情報(イベント会場登録時の任意入力)
- 広告識別子(IDFA/IDFV)
- OS・端末モデル・アプリバージョン等の技術情報

## 2. 利用目的

取得した情報は次の目的で利用します。

- 本人確認・認証
- サービスの提供および案内
- お気に入りのスポーツ・地域に関する通知配信
- ユーザー間のメッセージング機能の提供
- イベント・大会の運営および参加管理
- チーム所属管理・招待機能の提供
- 不正利用の防止、安全な運営、サービス品質改善
- 利用規約違反の調査および対応
- 広告の配信および効果測定
- 統計データの作成(個人を特定できない形式)

## 3. 第三者提供

運営者は、法令で認められる場合および以下の場合を除き、ユーザーの同意なく第三者に個人情報を提供しません。

- 法令に基づく場合
- 人の生命、身体または財産の保護のために必要がある場合
- 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
- 国の機関若しくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

## 4. 業務委託 (利用するクラウドサービス)

運営者は、サービス提供のため一部業務を以下の事業者に委託しています。

- Google LLC (Firebase Authentication, Firestore, Cloud Storage, Cloud Messaging, Cloud Functions, Maps SDK, Places API)
- Google LLC (Google Mobile Ads — 広告配信)
- Google LLC (Firebase Analytics — 利用状況解析)
- Apple Inc. (App Store — 配信および決済)

これらの委託先には、業務遂行に必要な範囲でのみ情報を取り扱うよう、適切な監督を行います。

## 5. 海外への移転

Firebase / Google Cloud Platform をはじめとした委託先の一部は、米国その他の国にデータを保管する場合があります。これらの国の個人情報保護制度は日本と同等とは限りませんが、運営者は委託先と適切なデータ処理契約を締結し、GDPR 等の国際的な水準に準じた取扱いを行います。

## 6. クッキー類似技術および広告識別子 (ATT)

本アプリでは、利用状況の計測および広告配信のため、Google Mobile Ads / Firebase Analytics 等が広告識別子等を取得します。

iOS 14.5 以降では、本アプリの起動時に「他社の App や Web サイトを横断してあなたのアクティビティを追跡することを許可しますか?」というダイアログ (App Tracking Transparency / ATT) が表示される場合があります。許可されない場合でも本サービスはご利用いただけますが、広告のパーソナライズが制限されることがあります。

広告のパーソナライズは、端末の「設定」アプリの「プライバシーとセキュリティ」→「Apple 広告」または「トラッキング」設定からいつでも変更できます。

## 7. チャット・メッセージング情報の取扱い

ユーザー間でやり取りされたメッセージは、Firebase Firestore に保存され、Firebase の通信時 (TLS) および保管時 (AES-256 等) の暗号化機能により保護されます。

ただし、これらはエンドツーエンド暗号化ではなく、運営者は Firebase 管理 SDK を通じて、適法な調査目的 (通報対応、不正利用調査、法令に基づく開示請求への対応等) のために、メッセージの内容を確認することがあります。通常運用において、運営者は個別のメッセージ内容を閲覧することはありません。

## 8. イベント参加情報の取扱い

ユーザーがイベントに参加すると、参加者一覧 (ニックネーム・プロフィール画像・性別) が、そのイベントの作成者および他の参加者に表示されます。これは本サービスの基本機能です。イベントから抜けた場合、参加履歴は当該イベントから削除されますが、運営者の統計データには含まれることがあります。

## 9. 通報・ブロック情報の取扱い

ユーザーが他のユーザーまたはコンテンツを通報した場合、通報者の ID、通報内容、対象ユーザー / コンテンツ ID 等が運営者の管理する Firestore に保存されます。通報を受けた側に通報者の情報が開示されることはありません。ブロック情報は、ブロックされた相手にはブロックされた旨は通知されませんが、メッセージの送信および新規チャットルームの作成が制限されます。

## 10. 保管期間

取得した個人情報は、利用目的の達成に必要な期間、また法令で求められる期間のみ保管し、不要となったものは速やかに削除または匿名化します。

- イベント関連データ: 開催から一定期間経過後、ユーザーの閲覧需要が低下した段階で順次削除されます。
- 退会したアカウント: 法的要件 (不正利用記録など) を除き、ご請求があれば速やかに削除します。また、不要となったデータは順次削除します。
- 不正利用記録: 再発防止のため最大 3 年間保管することがあります。

## 11. 安全管理措置

運営者は、取得した個人情報の漏洩、滅失または毀損を防止するため、通信の暗号化 (TLS)、アクセス権限の制御 (Firestore Security Rules)、アクセスログの監視等、必要かつ適切な安全管理措置を講じます。

## 12. 開示・訂正・削除等の請求

ユーザーは、自己の個人情報の開示・訂正・削除・利用停止等を希望する場合、お問い合わせ窓口 (borspo.support@gmail.com) までご連絡ください。

本人確認の上、原則として請求受付日から 30 日以内に対応します。技術的に対応に時間を要する場合や、本人確認に時間を要する場合は、その旨を別途ご連絡します。

## 13. 子どもの利用について

本サービスは原則として 13 歳以上を対象としています。13 歳未満の方は本サービスを利用できません。13 歳以上 18 歳未満の方が本サービスを利用する場合、保護者の同意のもとに利用していただきます。

## 14. プライバシーポリシーの変更

本ポリシーの内容は、法令の改正やサービスの変更に応じて随時改定する場合があります。重要な変更がある場合は、本アプリ内またはサービスサイト上で通知します。変更後も本サービスの利用を継続される場合、変更後のポリシーに同意したものとみなされます。
